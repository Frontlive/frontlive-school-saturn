## Konfiguracja

Klonowanie repozytorium:

```bash
git clone https://github.com/Frontlive/frontlive-school-neptun.git
```

Instalacja zalożności:

```bash
npm install
```

Odpalenie projektu:

```bash
npm run start
```urn
